# Pitchfork Anglocentrism Study

A quantitative content analysis examining whether Pitchfork's album reviews display Anglo-centric bias towards artists from Anglo-American countries.

## Data

- **Sample**: 270 album reviews, stratified across 9 genre categories (30 per genre)
- **Method**: Manual coding using MusicBrainz area field for country of origin
- **Tool**: R

## Hypotheses & Results

| Hypothesis | Result |
|---|---|
| H1: Non-Anglo artists < 25% of sample | Not supported (26.67%) |
| H2: >30% of non-Anglo artists tagged as "Global" | Partially supported |
| H3: No score difference between Anglo/non-Anglo artists | Supported |

## Files
